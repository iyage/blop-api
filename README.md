## Getting Started

please follow the following to build and run the program

### Prerequisites

Nodejs 18+
typescript

- npm

  npm install npm@latest -g

### Installation

1. Clone the repo

   ```sh
   git clone https://github.com/iyage/blop-api

   ```

2. Install NPM packages
   ```sh
   npm install
   ```
